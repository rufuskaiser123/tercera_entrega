Python

Python es uno de los lenguajes de programación más populares del mundo, utilizado para una amplia variedad de aplicaciones, desde el análisis de datos hasta el desarrollo de aplicaciones web. Este curso es una introducción completa al lenguaje de programación Python y te enseñará desde los conceptos básicos hasta cómo escribir programas más avanzados.

JavaScript

JavaScript es un lenguaje de programación muy popular que se utiliza en la mayoría de las páginas web modernas. Este curso comienza con los fundamentos de JavaScript y luego te lleva a través de conceptos más avanzados, como manipulación del DOM y programación orientada a objetos.

React

React es una biblioteca de JavaScript para construir interfaces de usuario. React es muy popular entre los desarrolladores web y se utiliza en muchas aplicaciones web modernas. Este curso te enseñará todo lo que necesitas saber sobre React, desde los conceptos básicos hasta cómo construir aplicaciones web avanzadas.
